# 系统核心架构说明

本项目 GUI 框架由以下三大核心系统支撑：

## 1. FocusManager (焦点与输入管理器)
- **单例模式**：全局统一的交互调度中心。
- **Z-Order 检测**：按注册顺序逆序检测碰撞，确保上方控件优先响应。
- **鼠标捕获 (Capture)**：按下鼠标时锁定操作控件，消除滑动时的悬停穿透。
- **状态分发**：自动管理 `m_isHovered` 和 `m_isFocused` 状态。

## 2. GuiTheme (主题系统)
- **配置中心**：在 `GuiTheme.h` 中统一定义所有组件的默认颜色、字体大小和间距。
- **一键换肤**：通过修改单例属性，可实现全局样式的同步更新。

## 3. GuiSkin (皮肤系统)
- **解耦渲染**：通过 `IGuiSkin` 接口，将业务逻辑与绘图代码彻底分离。
- **高度定制**：开发者可以实现自己的皮肤类，通过 `SetSkin()` 为特定控件或全局替换完全不同的视觉风格（如圆角、渐变、阴影等）。

## 4. GuiRenderer (绘图助手)
- **静态工具**：提供居中对齐文字、带边框矩形绘制等常用方法的封装，减少代码冗余。
