# Slider & IntSlider (滑块)

用于数值调节的线性控件，支持浮点数和整数。

## 特性
- **双向支持**：支持水平（Horizontal）和垂直（Vertical）方向。
- **多种输入模式**：
  - **点击跳转**：点击轨道直接将焦点移动到滑块而不改变数值，避免误触。
  - **拖动调节**：支持拖动滑块调节数值，在拖动过程中自动捕获鼠标，防止干扰其他控件。
  - **滚轮微调**：获取焦点时支持鼠标滚轮滚动微调。
  - **精准键盘控制**：
    - 使用方向键进行 +1 / -1 步长微调。
    - （浮点数滑块）按住 `Shift` 加方向键实现 +0.1 / -0.1 精确微调。
    - 使用 `Home` 和 `End` 键快速跳转到最小值和最大值。
  - **直接输入**：悬停时按住 `Ctrl + 鼠标左键点击`，即可唤出输入框，通过小键盘/数字键直接键入数值，`Enter` 键确认应用。
- **皮肤与高亮支持**：可自定义轨道、滑块及填充区域的视觉样式，在对焦状态下可维持高亮状态。

## 常用接口
- `Slider(Rectangle bounds, float min, float max, float value, ...)`: 浮点数滑块。
- `IntSlider(Rectangle bounds, int min, int max, int value, ...)`: 整数滑块。
